ระบบห้องสมุดดิจิทัล
งานนี้คืออะไร
ระบบห้องสมุดดิจิทัล (Digital Library System) เป็นเว็บแอปพลิเคชันที่พัฒนาด้วย Django สำหรับการจัดการห้องสมุดออนไลน์ระบบนี้สามารถสแกน-คืนหนังสือจองหนังสือและจัดการค่าปรับได้อย่างมีประสิทธิภาพ

หลักๆ
ค้นหาหนังสือ (Books) : ค้นหาเพิ่มเติมและดูรายละเอียดหนังสือ
ระบบบาร์โค้ด-คืน (Loans) : บาร์โค้ดหนังสือและติดตามสถานะการทำงานของบาร์โค้ด
ระบบจอง (Reservations) : จองหนังสือที่ไม่ว่างและดำเนินการจองคิว
ตะกร้าหนังสือ (Shopping Cart) : เพิ่มหนังสือหลายเล่มก่อนเปิดใช้งานทันที
ค่าปรับ (ค่าปรับ) : ติดตามและการจัดการค่าปรับกรณีคืนหนังสือล่าช้า
ผู้ใช้ระบบ (Users) : ลงทะเบียน เข้าสู่ระบบ และการลงทะเบียน
พนักงานผู้ดูแล : จัดการหนังสือผู้ใช้สามารถลงทะเบียน-คืน
วิธีการดำเนินโครงการ
เบื้องต้น
Python 3.8 MOT
pip (ตัวจัดการแพ็กเกจ Python)
สภาพแวดล้อมเสมือนจริง (แนะนำ)
ดาวน์โหลดและรัน
1. โครงการโคลน
git clone < repository-url > 
cd digital-library
2. สร้างสภาพแวดล้อมเสมือนจริง
python -m venv .venv
3. เปิดใช้งานสภาพแวดล้อมเสมือนจริง
ระบบปฏิบัติการ macOS/Linux:

source .venv/bin/activate
ระบบปฏิบัติการ Windows:

.venv \ Scripts \ activate
4. การพึ่งพาอาศัยกัน
pip install -r requirements.txt
5. จัดเตรียมเครื่องจักร
python manage.py migrate
6. สร้างผู้ดูแลระบบ (ถ้าต้องการ)
python manage.py createsuperuser
คำถามโปรแกรมถาม (ชื่อผู้ใช้ อีเมล รหัสผ่าน)

7. รันเบียร์
python manage.py runserver
8.เข้าสู่การใช้งานระบบ
ตรงและอลูมิเนียมที่:

หน้าหลัก : http://127.0.0.1:8000/
ผู้ดูแลระบบ : http://127.0.0.1:8000/admin/
โครงการต่างๆ
books/: แอปพลิเคชันจัดการหนังสือและตะกร้า
loans/: แอปพลิเคชันจัดการสแกน-คืน
reservations/: การสมัครจองหนังสือ
fines/: แอปพลิเคชันจัดการค่าปรับ
users/: โปรแกรมจัดการผู้ใช้และการตรวจสอบสิทธิ์
config/: คุณจังโก้ หลัก
templates/: เทมเพลต HTML
media/: รูปภาพปกของหนังสือ
docs/: เอกสารประกอบโครงการ

เทคโนโลยีที่ใช้
Django 4.2 : เว็บเฟรมเวิร์กหลัก
SQLite : เทคโนโลยี
Pillow : อุปกรณ์รูปภาพปกหนังสือ
HTML/CSS : เทมเพลตส่วนหน้า (Frontend templates)
